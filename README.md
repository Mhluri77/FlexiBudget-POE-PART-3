# FlexiBudget

FlexiBudget is a student project app made in Android Studio.  
It helps people manage their money by keeping track of expenses, showing reports, and giving rewards for good budgeting.  
The main idea is to make budgeting easy to understand and fun to use.

---

What the app can do?

- Add expenses like food, transport, or shopping.
- Save all expenses in a local Room database.
- Show graphs and charts so you can see where your money goes.
- Give rewards when you budget well.
- Simple design with easy screens.
